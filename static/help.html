Help / Frequently Asked Questions

What is the purpose of this website?

This website will (hopefully) provide information and answer questions the visitor may have regarding spacecraft, engines, space travel, exoplanets, etc.


How does the search feature work?

A search query is taken in from the user. The website parses the query for keywords, then parses the three APIs listed at the bottom of this page as appropriate for information relating to the search, and feeding it back to the user.


What if no keywords are found?

Searches not containing keywords will error out.


What are some examples of search queries?

The amount of APIs searched through depends on the keywords. Something like “how long to reach {{exoplanet}} with SpaceX Falcon 9” will work like this:

  • Search exoplanets API for the planet
  • Search Wikipedia API for the engine/rocket
  • Use the data gathered from the two above APIs to send an equation to Wolfram|Alpha
  • Return the result.

Meanwhile, a query like “stat[istic]s for {planet, rocket engine}” will just get a result from the exoplanets API or Wikipedia API respectively.



Have fun! This is very much a "try things out and see what happens" website.
